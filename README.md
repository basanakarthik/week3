## Project Description
A comprehensive contact manager using Python dictionaries and functions. Supports full CRUD, search, export, and persistence to JSON.

## How to run
1. Ensure Python 3.x is installed.
2. Run the program:
```
python contacts_manager.py
```
3. The program auto-loads `contacts_data.json` (sample provided). Use the menu to manage contacts.

## Files
- contacts_manager.py
- contacts_data.json (sample)
- test_contacts.py
- README.md
- requirements.txt
- screenshots/
- contacts_backup.json (created at runtime)

## Features
- Add, search, update, delete contacts
- Load/save JSON persistence
- Export to CSV
- Backup file support
- Phone & email validation
- Contact statistics
